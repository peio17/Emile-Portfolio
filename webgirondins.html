Automated Football Match Data Pipeline for a media

Project Overview

WebGirondins, a French sports media, used to collect player statistics manually during football matches. One analyst tracked performance on paper, limiting the volume of data gathered and the insights available. I proposed and built a complete data pipeline to automate this workflow and modernize the analysis experience.

My Role

  • Designed the data model for match events
  • Developed an Excel VBA interface for live data capture
  • Built a Power BI dashboard for visual analysis
  • Trained the team on how to use the system

Process

1. Excel VBA - Live Data Collection

I built a user-friendly interface in Excel where the analyst selects a player and clicks on actions (goal, pass, interception, duel, etc.). Each click automatically increments the right stat in the database behind the scenes.

2. Automated Data Structure

Every recorded action is structured into a dataset compatible with Power BI. This dataset creates a semi-automated pipeline from match event collection to visualization.

3. Power BI Visual Reporting

After each match, the analyst refreshes the Power BI report. The dashboard instantly updates match data, distributions, player performance indicators and comparisons.

Impact

  • Increase in volume of match statistics collected
  • Time to generate visuals reduced from 2 hours to ~2 minutes
  • Used live during post-match shows and published on WebGirondins
